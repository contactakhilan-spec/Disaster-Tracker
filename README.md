# Disaster-Tracker (Machine Learning Classifier)
An independent Python software project utilizing supervised learning classifiers to evaluate atmospheric telemetry and predict regional risk tiers.

## Tech Stack & Methodology
* **Language:** Python 3.x
* **Machine Learning:** `scikit-learn` (Trained a **Decision Tree Classifier** to map multivariate weather elements to explicit triage alert codes)
* **Data Management:** Native `json` parsing arrays for streaming telemetry data simulation.
